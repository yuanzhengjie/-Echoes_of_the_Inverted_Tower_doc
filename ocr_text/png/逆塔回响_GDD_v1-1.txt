《 逆 塔 回响 》 (Echoes of the Inverted Tower )
中 ny is
独立 游戏 策划 案 (GDD v1)
一 、 项 目 概述
项 目 性 质 : 独立 单机 游戏
类 型 : 魔 塔 / 数值 解 这 /回合 制 RPG
平台 : PC (Windows/mac0S) 优先 ， 预 留 移动 端 移植
核心 卖点 : 无 随机 战斗 ， 所 有 结果 可 计算 ; 资源 有 限 ， 决 策 驱动 推进 ; 强 复 盘 与 挑战 性 ©
二 、 核 心 设计 目 标
。 所 有 战斗 结果 可 预期 (无 随机 伤害 ) 。
。 资源 有 限 ， 每 一 步 存在 机 会 成 本 (钥匙 、HP、 人 金币、 路 线 ) 。
。 通关 依赖 理解 与 规划 ， 而 非 操作 练习 。
。 支持 多 结局 与 极限 挑战 ( 速 通 / 低 属性 通关 ) 。
三 、 基 础 系统 设计
3.1 玩家 属性
HP (生命 ) 、ATK (攻击 ) 、DEF (防御 ) 、GOLD (人 金钱 ) 、 钥 匙 ( 黄 / 蓝 / 红 ) 。
3.2 战斗 公式 (经典 魔 塔 )
玩家 每 回合 害 = max(ATK - HA DEF, 0)
敌人 每 回合 伤害 - max( 敌 人 ATK - DEF, 0)
回合 数 = ceil BLA HP / 玩家 每 回合 伤害 )
预计 损失 HP -= (回合 数 - 0 x 敌人 每 回合 伤害 (BAMA)
若 玩家 每 回合 嘎 ~ 0， 则 不 可 战胜 。
