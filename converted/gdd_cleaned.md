三 、 基础系统设计
### 3.1 玩家属性
HP (生命 ) 、ATK (攻击 ) 、DEF (防御 ) 、GOLD (人金钱 ) 、 钥匙 ( 黄 / 蓝 / 红 ) ©
### 3.2 战斗公式 (经典魔塔 )
玩家每回合害 = max(ATK - HA DEF, 0)
敌人每回合伤害 = max( 敌人 ATK - DEF, 0)
回合数 = ceil( 敌人 HP / 玩家每回合伤害 )
预计损失 HP = (回合数 - 1) x 敌人每回合伤害 (SSM)
PRB AE = 0， 则不可战胜 。
